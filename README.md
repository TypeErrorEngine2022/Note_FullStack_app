# Note FullStack app

Frontend: React  
Backend: NestJS  

## Setup
please clone all the submodules
```bash
pnpm i
```


## Frontend usage
```bash
pnpm dev
```
Open http://localhost:5173/

## Backend usage
```bash
pnpm start:dev
```
